# 🎵 Spotify Clone

A responsive Spotify-inspired music streaming website built using HTML and CSS.

## 🚀 Features

- Responsive Navigation Bar
- Sidebar with Library Section
- Search Bar
- Playlist Card
- Language Button
- Trending Songs Section (In Progress)
- Modern Spotify-inspired UI

## 🛠️ Tech Stack

- HTML5
- CSS3
